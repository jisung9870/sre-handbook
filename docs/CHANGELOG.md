## [1.0.0] - 2026-03-28

### Added

- **문서 뷰어** — 28개 SRE 가이드 HTML 문서, 원본 CSS(카드 그리드, 코드 블록, 다이어그램) 보존 렌더링
- **카테고리 사이드바** — 9개 카테고리, 접기/펼치기, 현재 문서 액센트 컬러 하이라이트
- **React 탭 재구현** — 원본 탭 구조(`data-t`/`data-tab`) 추출, URL `?tab=` 동기화
- **Cmd+K 전문 검색** — Fuse.js 퍼지 검색, 카테고리별 그룹핑, 키보드 완전 지원 (↑↓, Enter, ESC)
- **북마크** — 문서/탭/섹션 단위, IndexedDB(Dexie.js) 영속 저장, 북마크 목록 페이지
- **메모** — 탭별 인라인 에디터(Cmd+Enter 저장, ESC 취소), IndexedDB 영속 저장, 메모 목록 페이지
- **다크 모드** — OS 기본값 연동 + 수동 전환, FOUC 없는 즉시 적용, 28개 문서 CSS 변수 오버라이드
- **PWA** — Service Worker 오프라인 캐싱 (docs 30일, data 7일, 폰트 1년), 홈 화면 추가 지원
- **새 문서 자동화** — `docs-config.yaml` 기반, 미등록 파일 경고, `pnpm extract` 한 번으로 반영
- **반응형 레이아웃** — 모바일 슬라이드 사이드바, 데스크톱 고정 260px 사이드바
- **페이지 전환 애니메이션** — 라우트 변경 시 slide-up-fade, 검색 모달 fade-in
- **브레드크럼 네비게이션** — 문서 페이지 상단 `홈 > 카테고리 > 문서명`
- **홈 통계 요약** — 문서 수, 북마크 수, 메모 수 실시간 표시
- **최근 본 문서** — 최근 10개, 카테고리 컬러 dot + 상대 시간 표시
